# RoadScope AI: Intelligent Pothole Detection & Mapping

RoadScope AI is a comprehensive system for automating road safety inspections. It uses computer vision (YOLOv8) to detect potholes from video footage, synchronizes detections with GPS data, and visualizes the results on an interactive map dashboard.

## 📂 Project Structure

```
ScopePotholeDetection/
├── data/                  # Data storage
│   ├── dataset/           # Training images and labels for YOLO
│   ├── gps_logs/          # GPX/CSV GPS tracks
│   └── raw_videos/        # Input video footage
├── models/                # Trained YOLOv8 models (.pt)
├── src/
│   ├── detection/         # Phase 1 & 2: Computer Vision
│   │   ├── extract_frames.py  # Convert video to dataset
│   │   ├── train.py           # Train YOLOv8 model
│   │   └── detect.py          # Run inference on video
│   ├── mapping/           # Phase 3: Geospatial Processing
│   │   ├── parse_gpx.py       # Parse GPX files to CSV
│   │   ├── sync_coords.py     # Merge detections with GPS
│   │   └── snap_to_road.py    # Google Roads API integration
│   └── dashboard/         # Phase 4: Next.js Web App
│       ├── app/               # Frontend Logic
│       └── components/        # Map & Stats Components
└── requirements.txt       # Python dependencies
```

## 🚀 Getting Started

### 1. Python Environment (Detection & Mapping)
**Prerequisites:** Python 3.10+

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Note: PyTorch is large, download on a stable connection).*

2.  **Run Inference (Example):**
    ```bash
    python src/detection/detect.py --video data/raw_videos/road_trip.mp4 --model models/yolov8n.pt
    ```

### 2. Dashboard (Visualization)
**Prerequisites:** Node.js 18+

1.  **Navigate to Dashboard:**
    ```bash
    cd src/dashboard
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Configure API Key:**
    *   Rename `.env.local.example` to `.env.local`
    *   Add your `NEXT_PUBLIC_GOOGLE_MAPS_KEY`
4.  **Run Locally:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000).

## 🛠️ Workflow

1.  **Capture Data:** Record video (`.mp4`) and GPS (`.gpx`) simultaneously.
2.  **Detect:** Run `detect.py` to generate `detections.csv`.
3.  **Map:**
    *   Run `parse_gpx.py` to get `gps.csv`.
    *   Run `sync_coords.py` to combine them into `mapped_potholes.csv`.
    *   (Optional) Run `snap_to_road.py` for precision.
4.  **Visualize:** Upload `mapped_potholes.csv` to the Dashboard.

## ☁️ Deployment
*   **Frontend:** Deploy `src/dashboard` to [Vercel](https://vercel.com).
*   **Backend/Training:** Run on a GPU-enabled machine (local PC or Colab).
